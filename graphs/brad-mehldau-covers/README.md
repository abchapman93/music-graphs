A graph centered on Brad Mehldau's engagement with the rock, pop, and folk songbook. Scope is limited to Mehldau as leader or co-leader. Albums devoted to a single artist or theme (e.g., *Your Mother Should Know*, *Ride Into the Sun*) appear as album nodes when ≥80% of tracks fit; otherwise individual track nodes are used. Each Mehldau recording is linked to the original song and original artist.
